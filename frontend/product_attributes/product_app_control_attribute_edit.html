Product App Control

Attribute terms can be assigned to products and variations.

Note: Deleting a term will remove it from all products and variations to which it has been assigned. Recreating a term will not automatically assign it back to products.

Add new App Control

The Name is how it will appear on the site.

The “slug” is the URL-friendly version of the name. Must be all lowercase and contain only letters, numbers, and hyphens.

The description is only used in the backend.
